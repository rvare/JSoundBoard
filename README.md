# JSoundBoard

A simple soundboard written in Java using Java Swing.

## Features

JSoundBoard allows you to load up to nine different sound clips to play whenever you want.

### Currently Supported Audio Formats

- Wave (`.wave`)

## Help

To get help within the application, go to `help` then click on `Documentation`.

You can also find more help in the `/docs` directory.
